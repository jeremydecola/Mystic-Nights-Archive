Quick Start Tutorial

How to Set Up Network Configuration for Multiplayer

Begin the ritual.
Power on your PlayStation 2 or PCSX2.
Memory Card must be inserted.
 Follow each step below and match your settings to the screenshots:
Step 1: Select 'NETWORK GAME' from the main menu.
Step 2: Press Circle button (OK) to acknowledge message.
Step 3: Navigate down to select "Manage Network Settings".
Step 4: Press Circle button (OK) to enter Network Settings Manager.
Step 5: Wait for Network Settings Manager to finish loading...
Step 6: Select the first menu option (Create New Configuration).
Step 7: Select the Memory Card you wish to save to.
Step 8: Select the left option in the pop-up (YES).
Step 9: Press Circle button (OK) to acknowledge message.
Step 10: Press "RIGHT" directional button to go to the next screen.
Step 11: Press Circle button to select "SCE/Ehernet (Network Adapter) from the list. It will turn blue. Press "RIGHT" directional button to go to the next screen.
Step 12: Press Circle button (OK) to acknowledge message.
Step 13: Chose the bottom option with Circle. Press "RIGHT" directional button to go to the next screen.
Step 14: Chose the top option with Circle if you would like to use DHCP [chose this if you don't know what you are doing] or the bottom option if you want to set a Static IP. Press "RIGHT" directional button to go to the next screen.
Step 15: Chose the top option with Circle if you would like to set up DNS automatically [chose this if you don't know what you are doing] or the bottom option if you want to set a DNS manually. Press "RIGHT" directional button to go to the next screen.
Step 16: Press Circle if you would like to change the name of your configuration. The default name is "Setting 1". Press "RIGHT" directional button to go to the next screen.
Step 17: Press Circle button (OK).
Step 18: A pop-up will appear.
Step 19: Press Circle button (OK).
Step 20: Select the left option in the pop-up (YES).
Step 21: Press Circle button.
Step 22: Press Circle button (CONFIRM).
Step 23: Press Circle button (OK).
Step 24: Select the left option in the pop-up (YES).
Step 25: Press X button to Quit out of Network Settings Manager.
Step 26: Select the left option in the pop-up.
Step 27: You're ready to log in now!
